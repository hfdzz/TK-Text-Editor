# TK-Text-Editor

Simple text editor made with Tkinter python library.

## Feature

- Open/save file
- Text editting

## Planned Feature

- Unsaved change warning
- Undo/redo
- Open multiple file

## Known Bugs

- Saving deleted file  
    Saving deleted file will create a new file with the same name as previous file name.
    > should've prompt a save as dialog.
